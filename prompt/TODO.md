# 웹 네임드(Neimd) QA 테스트 체크리스트

## 📋 프로젝트 개요
- **프로젝트명**: Neimd (디지털 명함 생성 웹 애플리케이션)
- **기술 스택**: Next.js 15.2.4, React 19, TypeScript, Tailwind CSS, Zustand, Supabase
- **작성일**: 2025-09-08
- **버전**: 1.0.0

## 1. 🔐 인증 시스템 (Authentication)

- [x] **소셜 회원가입**
  - [x] Google OAuth 회원가입 플로우
  - [x] Kakao OAuth 회원가입 플로우
