# 🚀 사용자/관리자 완전 분리 체크리스트

## 📋 프로젝트 개요
- **목표**: 한 도메인에서 `/user`, `/admin` 경로로 완전 분리
- **핵심**: 같은 이메일로 사용자/관리자 각각 가입 가능
- **기술**: 별도 Supabase 프로젝트 + 경로 기반 클라이언트 분기
